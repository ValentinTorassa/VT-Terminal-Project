# Ghostty Shortcuts

## Window & Tab

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+N` | New window |
| `Ctrl+Shift+T` | New tab |
| `Ctrl+Shift+W` | Close tab/split |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Ctrl+Shift+1..9` | Go to tab N |

## Splits

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+O` | Split horizontal |
| `Ctrl+Shift+E` | Split vertical |
| `Ctrl+Shift+Arrow` | Navigate splits |
| `Ctrl+Shift+Alt+Arrow` | Resize split |
| `Ctrl+Shift+Z` | Toggle zoom split |

## Text

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+C` | Copy |
| `Ctrl+Shift+V` | Paste |
| `Ctrl+Shift+=` | Increase font size |
| `Ctrl+Shift+-` | Decrease font size |
| `Ctrl+Shift+F` | Search |

## Other

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+,` | Open config file |

## macOS Variants

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New window |
| `Cmd+T` | New tab |
| `Cmd+W` | Close tab/split |
| `Cmd+D` | Split right |
| `Cmd+Shift+D` | Split down |
| `Cmd+C` | Copy |
| `Cmd+V` | Paste |
| `Cmd+=` / `Cmd+-` | Font size |
