# Git Shortcuts

## Aliases (defined in dotfiles)

| Alias | Command |
|-------|---------|
| `gs` | `git status` |
| `gc` | `git commit` |
| `gp` | `git push` |
| `gpl` | `git pull` |
| `gsw` | `git switch` |
| `lg` | Launch lazygit |

## Git Config Aliases

| Command | Action |
|---------|--------|
| `git unstage` | Unstage files |
| `git undo` | Soft reset last commit |
| `git amend` | Amend last commit (keep message) |

## Lazygit Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space` | Stage/unstage file |
| `a` | Stage all |
| `c` | Commit |
| `p` | Push |
| `P` | Pull |
| `b` | Branch menu |
| `?` | Help |
| `q` | Quit |
