# Terminal Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+A` | Move cursor to beginning of line |
| `Ctrl+E` | Move cursor to end of line |
| `Ctrl+U` | Delete from cursor to beginning of line |
| `Ctrl+K` | Delete from cursor to end of line |
| `Ctrl+W` | Delete word before cursor |
| `Ctrl+L` | Clear screen |
| `Ctrl+C` | Cancel current command |
| `Ctrl+D` | Exit shell / EOF |
| `Ctrl+R` | Search command history (fzf) |
| `Ctrl+T` | Fuzzy find file (fzf) |
| `Alt+C` | Fuzzy cd into subdirectory (fzf) |
| `Ctrl+Shift+C` | Copy selection (terminal emulator) |
| `Ctrl+Shift+V` | Paste from clipboard (terminal emulator) |
| `Tab` | Autocomplete command/path |
